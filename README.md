clean-clipboard
===============

Windows application to clean clipboard on timeout after pressing keyboard keys combination.

Requires:
```
CMake 2.6
MinGW-g++ 4.8.2
DInput
```

Build:
```
cmake -DCMAKE_BUILD_TYPE=Release -G "MinGW Makefiles" CMakeLists.txt
mingw-make
```
